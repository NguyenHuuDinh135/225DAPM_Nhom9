# Thư mục Public - Chứa ảnh tĩnh

## Hướng dẫn thêm ảnh cho trang đăng nhập:

1. **Đặt ảnh vào thư mục này** với tên: `login-background.jpg` (hoặc `.png`, `.webp`)
   
2. **Vị trí**: `frontend/apps/web/public/login-background.jpg`

3. **Kích thước khuyến nghị**: 
   - Chiều rộng: 800-1200px
   - Chiều cao: 600-900px
   - Tỷ lệ: 4:3 hoặc 16:9

4. **Ảnh sẽ hiển thị**: Bên phải form đăng nhập (trên desktop)

## Các file ảnh khác có thể đặt ở đây:
- `favicon.ico` - Icon trang web
- `logo.png` - Logo công ty
- `og-image.jpg` - Ảnh chia sẻ mạng xã hội
